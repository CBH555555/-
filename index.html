📤 Datei für GitHub exportieren

Exportiert alle Fragen UND Bilder in eine neue HTML-Datei die du direkt auf GitHub hochladen kannst.

So lädst du es auf GitHub hoch:
1. Gehe auf github.com → kostenloses Konto erstellen
2. Oben rechts auf + → New repository
3. Name eingeben z.B. unser-spiel → Public → Create
4. Klicke auf uploading an existing file
5. Exportierte HTML-Datei reinziehen → Commit changes
6. Gehe zu Settings → Pages → Branch: main → Save
7. Dein Link: deinname.github.io/unser-spiel
